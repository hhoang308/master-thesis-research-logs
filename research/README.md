# Research Layout

This directory is organized by research role:

- `pocs/`: reproduced proof-of-concepts.
- `crashes/`: findings that still need triage or confirmation.
- `targets/`: target source trees and local builds by target/version.
- `seeds/`: collected, minimized, and dictionary seed material.
- `repos/`: external open-source repositories and submodules.
- `schema/`: structure-aware grammars, serializers, and verifiers.
- `harnesses/`: harnesses grouped by target/version/engine.
- `experiments/`: run records and scripts; bulky fuzzing outputs are ignored.
- `baselines/`: baseline fuzzing/reference run records.
- `docs/`: research notes and thesis material.
