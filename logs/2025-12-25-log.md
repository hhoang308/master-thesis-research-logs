## Tasks
##### 1. Clone and build Nautilus
Local install because I doesn't have root permission
- `curl https://sh.rustup.rs -sSf | sh`
- `source "$HOME/.cargo/env"`
- `cargo run --bin generator -- -g grammars/javascript_new.py -t 100`
    - `cargo run`: compile và run
    - `--bin generator`: chạy riêng phần tên là generator
    - `--`: ngăn cách giữa tham số truyền cho cargo và tham số truyền cho chương trình generator chuẩn bị build và run
    - `-g <file>`: chỉ định file grammar
    - `-o <folder>`: chỉ định output folder
    - `-n <number>`: chỉ định số lượng file sinh ra
    - `-t <time>`: thời gian chạy sinh dữ liệu (đơn vị mili giây)
##### 2. Generates some JavaScripts files
Nautlius chứa nhiều thành phần, trong đó thành phần tạo file tên là generator, nó chỉ dùng để test xem grammar có dùng được hay không thôi. Nếu muốn tạo thành nhiều file thì phải sử dụng nautilus hoàn chỉnh.
## Takeaway
1. Cargo là package manager và build system của Rust, nó tương tự với việc sử dụng pip để tải các dependency rồi ngay lập tức dùng lệnh `python example.py`. Do đó sẽ không thấy bước build và thực thi vì cargo làm hộ hết rồi.
